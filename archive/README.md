- Current generated/reproduction outputs live outside `archive/`.
- Manuscript versions in Rounds 46-49 are historical.
- `archive/round50` preserves the originally delivered Round 50 package, including the archived import-time endpoint behavior documented in the Round 51 precision finding.
- The precision-fixed implementation and outputs are located in `certificates/round50/round50_krawczyk_certifier_precision_fixed.py` and `results/round50_precision_fixed/`.
