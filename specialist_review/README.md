# Specialist review package

Begin with `SPECIALIST_REVIEW_REQUEST.md`, then read the manuscript and `THEOREM_CLAIM_AUDIT.csv`.

The `evidence/` directory contains compact machine-readable summaries. The complete source, interval boxes, regression code and historical artifacts are in the parent Round 52 repository.

This package asks for adversarial scientific review. It does not ask the reviewer to endorse novelty or correctness without checking the stated evidence.
