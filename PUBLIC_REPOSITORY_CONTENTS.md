# Public repository contents

This public package intentionally includes scientific and reproducibility artifacts while excluding journal-submission administration.

## Included

- current manuscript source, figures, and PDF;
- full hierarchy nodes, weights, and alternation data;
- interval-certification code and frozen certificates;
- independent author-produced verification code;
- scientific audit reports and specialist-review materials;
- valid research history needed to trace the development;
- MIT license, citation metadata, locked dependencies, checksums, and tests.

## Intentionally excluded

- cover letters;
- corresponding-author telephone number;
- Editorial Manager field sheets;
- suggested-reviewer forms;
- draft competing-interest forms;
- internal submission blockers and administrative notes;
- LaTeX build caches and Python bytecode caches.

These exclusions do not remove any result required to inspect or reproduce the scientific claims.
